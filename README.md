# 한국무예위원회 (KMC) 공식 웹사이트

**Korea Martial Arts Committee** — TAFISA 국가회원 · UIA 국제기구 등록 · IOC·UNESCO 공인

---

## 소개

한국무예위원회(KMC)는 TAFISA(세계생활체육연맹) 국가회원으로 승인된 대한민국 대표 전통무예 통합단체입니다.
본 저장소는 KMC 공식 홈페이지의 소스코드입니다.
